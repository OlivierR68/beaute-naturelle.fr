\mbox{\hyperlink{class_smarty}{Smarty}} versions 3.\+1.\+11 or later are now on github and can be installed with Composer.

The \char`\"{}smarty/smarty\char`\"{} package will start at libs/.... subfolder.

To get the latest stable version of \mbox{\hyperlink{class_smarty}{Smarty}} 3.\+1 use\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{"require": \{}
\DoxyCodeLine{    "smarty/smarty": "\string~3.1"}
\DoxyCodeLine{\}}
\end{DoxyCode}


in your composer.\+json file.

To get the trunk version use\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{"require": \{}
\DoxyCodeLine{    "smarty/smarty": "\string~3.1@dev"}
\DoxyCodeLine{\}}
\end{DoxyCode}


For a specific version use something like\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{"require": \{}
\DoxyCodeLine{    "smarty/smarty": "3.1.19"}
\DoxyCodeLine{\}}
\end{DoxyCode}


P\+H\+P\+Unit test can be installed by corresponding composer entries like\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{"require": \{}
\DoxyCodeLine{    "smarty/smarty-\/phpunit": "3.1.19"}
\DoxyCodeLine{\}}
\end{DoxyCode}


Similar applies for the lexer/parser generator.


\begin{DoxyCode}{0}
\DoxyCodeLine{"require": \{}
\DoxyCodeLine{    "smarty/smarty-\/lexer": "3.1.19"}
\DoxyCodeLine{\}}
\end{DoxyCode}


Or you could use\+:


\begin{DoxyCode}{0}
\DoxyCodeLine{"require": \{}
\DoxyCodeLine{    "smarty/smarty-\/dev": "3.1.19"}
\DoxyCodeLine{\}}
\end{DoxyCode}


Which is a wrapper to install all 3 packages.

Composer can also be used for Smarty2 versions 2.\+6.\+24 to 2.\+6.\+30.
